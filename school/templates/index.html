Projeto Escola

Um projeto Django

Descrição Básica

Este projeto é um sistema de gerenciamento de uma escola, com área de administração personalizada para que o registro escolar possa cadastar alunos e professores, além de definir horários e disciplinas.

Os professoeres possuem um dashboard personalizado, permitindo listar todas as disciplinas em que ele leciona aulas, alem de cadastrar avaliações e lançar notas aos alunos.

Os alunos por sua vez, contam com um dashboard que os permitem ver as disciplinas, e as notas lançadas de cada uma das respectivas avaliações.
